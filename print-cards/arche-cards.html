ARCHÉ

Ce que vous tenez

Une carte. Pas un ticket. Pas un code promotionnel. Une clé.

Le QR code au dos ouvre une expérience de marche dans Paris. Trois quêtes. Sept symboles à chasser. Des traces laissées par d'autres marcheurs.

Comment imprimer

  • Papier recommandé : 300gsm, ivoire ou blanc cassé
  • Découper selon les lignes
  • Plier si format double face

Philosophie

ARCHÉ n'optimise pas l'engagement. Il enregistre le passage.

Une carte = une personne = une trace unique dans la ville.

Les cartes suivantes

PS-0001, PS-0002, PS-0003

Chaque carte est unique. Une fois activée, elle appartient à son marcheur.
